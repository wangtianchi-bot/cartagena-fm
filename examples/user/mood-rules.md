# 情绪规则（mood-rules.md）—— 示例模板

> 「我说累了的时候你放什么」这类 if-then 规则，5–8 条即可。
> 复制到 `user/mood-rules.md` 并换成你自己的规则。到 V1.2 自进化闭环上线后，
> 电台会根据你的切歌/听完/点歌行为提议修订这些规则（确认后才合入）。

## 显式情绪指令（你说出口时）

- 你说"累了" → 切到更轻柔舒缓的曲目，DJ 一句话确认，随后少说话
- 你说"专注"/"工作了" → 低人声干扰、偏器乐或纯律动，停止闲聊式串场
- 你说"放松"/"歇会儿" → 城市民谣、慢板、温柔人声
- 你说"振奋一下"/"提提神" → 节奏上扬的曲目
- 你说"emo 了"/"有点丧" → 淡淡情绪的歌，但避免致郁；DJ 只陪伴不说教
