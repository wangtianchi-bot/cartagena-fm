# 📻 私人 AI 电台 Agent

> 一个跑在自己电脑上的中文私人电台：知道你的品味、看得到你的日程和窗外的天气，在歌与歌的接缝处像老朋友一样开口，而且越听越懂你。

不是又一个音乐播放器，而是一个 **Agent**：一位有人格的中文 DJ 为你一个人开台——按你的品味选歌、在歌缝里串场、能用语音热线和你聊天，并随着你的收听行为慢慢长出对你的理解。全部数据留在本机。

## 灵感来源

- **mmguo 原版 Claudio**（抖音博主，未开源）：首创「把多年歌单蒸馏成 AI 电台」
- **开源复刻版 [bingyanglu/Claudio-FM](https://github.com/bingyanglu/Claudio-FM)**：复刻了工程骨架并验证了核心模式

本项目在两者基础上重做，补回了原版有、复刻版丢失的「陪伴感」模块（日程/天气感知），并新增了两版都没有的**品味自进化闭环**。

## 核心特性

| 能力 | 说明 |
|------|------|
| 🎙 **两段式选歌**（防幻觉的根） | LLM 先选歌 → 系统逐曲验证真实可播 → 只把确认歌单回喂 LLM 写口播。DJ 报出口的每首歌都播得出来。 |
| 🔊 **接缝串场** | DJ 只在歌尾 ≤15s 压混或歌间说话，绝不压人声段。满串场模式像真 FM：每首歌头尾都有点评与推荐。 |
| ☎️ **语音热线 + 情感陪聊** | 按住说话像打进电台热线；流式逐字上屏、逐句出声；倾听型人格陪你聊到落点，再送一首贴合对话的歌。 |
| 🌦 **天气感知**（F10） | Open-Meteo 免密钥；下雨自动偏向空灵曲目，串场可提一句窗外的雨。 |
| 📅 **日程感知**（F9） | 读当日日历，重要日程前自动进入低打扰模式。 |
| ⏰ **定时主动节目**（F11） | 07:00 拟当日节目计划、09:00 早间自动开播、整点 vibe check、日程临近提醒。 |
| 🌱 **品味自进化**（F12） | 被动采集切歌/听完/点歌行为 → 每周生成《品味观察周报》→ **你确认后**才合入品味档案。越听越懂你。 |

## 三个值得一讲的设计决策

1. **两段式生成防幻觉**：LLM 在管线里出现两次，中间隔一道「现实验证」关卡。口播阶段的 LLM 只看得到已确认可播的歌单，所以它说出口的歌必然存在——一次「报了不存在的歌」就击穿信任，这是 P0 红线。
2. **接缝即交互**：DJ 说话被严格调度到歌尾窗口或歌间，「不打扰音乐」是电台体验的底线。语音点歌是一通「打进热线」的仪式，不是一个搜索框。
3. **自进化闭环 + 用户主权**：行为信号 → 周报提案 → 你确认 → 合入档案。Agent 只能**提案**，品味档案（Markdown，人可读可改）只有你点了「采纳」才会变一个字。

## 三层记忆架构

大模型本身是无状态的——「记住你」靠的是模型之外自建的记忆，每次调用拼进 prompt：

- **① 工作记忆**（内存）：当前队列、上次串场长度 → DJ 自控节奏
- **② 情节记忆**（SQLite）：播放/对话/行为信号 → 去重、近期上下文
- **③ 语义记忆**（Markdown）：`user/taste.md` 等 → 关于「你是谁」的稳定结论

长期记忆刻意用人可读可改的 Markdown，而非藏进数据库——这是「用户主权」的工程落地。

## 快速开始

### 1. 准备账号与密钥（自带钥匙 BYO Keys）

| 用途 | 怎么获取 |
|------|----------|
| LLM（DJ 大脑） | [platform.deepseek.com](https://platform.deepseek.com) 创建 API Key |
| TTS / ASR（语音） | [火山引擎](https://console.volcengine.com) 开通豆包语音合成 + 流式语音识别 |
| 音乐源 | 网易云账号（启动时扫码登录本地 sidecar） |
| 天气（可选） | 无需密钥（Open-Meteo），填城市坐标即可 |

### 2. 安装与配置

```bash
git clone <this-repo> && cd cartagena-fm
npm install
brew install yt-dlp ffmpeg          # 备用音乐源（可选）

cp .env.example .env                 # 填入你的密钥
cp -r examples/user ./user           # 换成你自己的品味/作息/情绪规则
# 按需编辑 prompts/dj-persona.md（DJ 人格，可定制成你喜欢的样子）
```

### 3. 运行

```bash
npm run netease                      # 终端 A：起网易云 sidecar，首次扫码登录
node scripts/netease-login.js        # （如搜不到歌）清洗并落盘登录态
npm start                            # 终端 B：起电台 → http://localhost:8080
```

打开浏览器点「开台」即可。

### 4.（可选）macOS 一键启动 App

```bash
cd assets && ./build-app.sh          # 在桌面生成「电台.app」，双击即起服务并打开浏览器
```

## 配置说明

- `user/taste.md` `user/routines.md` `user/mood-rules.md`：你的品味/作息/情绪规则（**个人数据，已 gitignore**，模板见 `examples/user/`）
- `prompts/dj-persona.md`：DJ 人格档案，可自由改写
- `.env`：密钥与开关（**已 gitignore**）
- 编排数值（补歌间隔、接缝窗口、压混音量等）：`src/config.js` 的 `rules`

## 隐私

全部服务跑在本机，无公网暴露；播放历史/对话/行为信号只存本地 SQLite；API 密钥走 `.env` 不入库不入 git。日程/天气摘要会进入 LLM 请求的 prompt（需你知情）。
